Introductory examples for using bidir_submatrix can be found here.

1) run_1_dataset.py: 
After configuring the directories, you can run 1 dataset at a time by calling for example: 

python run_1_dataset.py --round 0 --dataset dorothea

2) run_all_datasets.py:
run all datasets by calling run_1_dataset.py recursively. Only for dev use (because not all datasets are available here).
